# 🌐 Mi Portfolio Web

Desarrollo de mi sitio web profesional basado en WordPress.

## Estructura

```
wp-content/
├── themes/      # Temas personalizados
├── plugins/     # Plugins del proyecto
└── mu-plugins/  # Must-use plugins
```

## Notas

- ⚠️ No sincronizar `wp-content/uploads/` (imágenes y medios)
- Usar Git LFS para archivos binarios grandes si es necesario
